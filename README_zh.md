# Fast ADB - 快速执行 ADB 命令的效率工具

Fast ADB 是一款专为 Android 开发者和测试人员打造的工具应用。
告别手动输入繁琐命令，快速执行常用 ADB 操作，提高开发与调试效率，使用 Android 设备文件管理系统，便捷操作设备文件。

## 下载:
MacOS :&nbsp;&nbsp;&nbsp;[Fast.ADB-1.4.1.dmg](https://github.com/Maxyjy/FastADB/releases/download/v1.4.1/Fast.ADB-1.4.1.dmg)

Windows : [Fast.ADB_1.4.1.zip](https://github.com/Maxyjy/FastADB/releases/download/v1.4.1/Fast.ADB_1.4.1.zip)

## 功能：
🚀 一键执行 ADB 命令：无需记忆和手动输入命令，点击即可执行。

✏️ 支持自定义ADB命令集快捷按钮：用户可添加/编辑自己的常用命令（别担心，已经内置了一套常用的默认ADB命令集）

📤 ADB命令集支持以JSON格式的导入/导出：方便将常用命令配置导入导出，导出给其他人或导入别人的配置使用。

📁 文件管理功能：支持浏览、管理Android设备中的文件与目录。

🕒 包含常用的Unix时间戳转换、JSON格式化、Base64编解码小工具。

## 跨平台支持
基于 Kotlin MultiPlatform 开发，支持MacOS、Windows、Linux。

（Linux：需自行在Linux系统中构建KMP环境，运行该代码，打包deb来安装，开发者没有Linux设备来进行打包)
